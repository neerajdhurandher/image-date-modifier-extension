# image-date-modifier-extension
This extention help to update the date of images &amp; video in google photos to maintain the timeline

## Quick start (Chrome/Edge)
1. Open the Extensions page (Chrome: `chrome://extensions`, Edge: `edge://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this project folder.
4. Visit `https://www.google.com/` to see the bottom-right popup.

## Project structure
- [manifest.json](manifest.json) — Extension manifest (MV3).
- [src/content.js](src/content.js) — Injects the on-page popup and script actions.
- [src/styles.css](src/styles.css) — Styles for the round popup UI.
- [REQUIREMENTS.md](REQUIREMENTS.md) — Functional requirements.
